

ForefrontAI
=====================

该页面介绍了如何在LangChain中使用ForefrontAI生态系统。分为两个部分：安装和设置，以及指向特定ForefrontAI包装器的参考。

安装和设置
---------------

- 获取ForefrontAI API密钥，并将其设置为环境变量（`FOREFRONTAI_API_KEY`)

包装器
---------------

**LLM**

ForefrontAI LLM可通过以下代码进行访问：

```
from langchain.llms import ForefrontAI
```
